# Hospital Management System V2

A full-stack **Hospital Management System**- HMS, built as part of the **Modern Application Development II** course.  
The application allows **Admins, Doctors, and Patients** to interact through a web interface for managing appointments, treatments, and hospital records.

The system demonstrates the use of **Flask APIs, VueJS frontend, Redis caching, and Celery background jobs**.

---

# Tech Stack

## Backend
- Flask (REST API)
- SQLite (Database)
- SQLAlchemy (ORM)
- Flask Security (Authentication)
- Redis (Caching)
- Celery + Redis (Background Jobs)
- Flask-Mail (Email services)
- Maihog

## Frontend
- Vue.js (Options API)
- Vue Router
- Bootstrap
